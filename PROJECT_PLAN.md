# ExpenseOS

## Vision

ExpenseOS is a modern personal finance management application that helps users track income, expenses, and financial habits through a beautiful and intuitive dashboard.

---

## Problem Statement

Many people struggle to understand where their money goes. ExpenseOS helps users record transactions, categorize expenses, visualize spending, and make better financial decisions.

---

## MVP Features

- User Authentication
- Dashboard
- Add/Edit/Delete Transactions
- Categories
- Analytics
- Profile

---

## Future Features

- Budget Goals
- AI Insights
- OCR Receipt Scanner
- Recurring Transactions
- Export PDF
- Multi Currency
